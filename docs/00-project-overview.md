# 00. 项目概览

## 项目定位

“迹”是一款企业桌面协作应用。第一阶段重点不是做大而全的 IM，而是先把企业员工每天都会用到的 Memo、通知、快捷记录、桌面提醒、权限共享做好。

核心体验：

- 应用常驻后台，有系统托盘图标。
- 可通过全局快捷键唤出主界面或快速 Memo 小窗口。
- Memo 支持可视化编辑、Markdown 源码、图片、Mermaid 图形。
- Memo 可以共享给相关人，并区分只读和可编辑权限。
- 其他人更新和共享 Memo 时，当前用户收到系统通知。
- 通知中心可查看 Memo 通知和未来其他系统通知。

## 与普通 Web 应用的区别

“迹”是桌面应用，不是普通网页：

- 没有“刷新页面”作为主要交互。
- 需要自动保存和草稿保护。
- 需要托盘、全局快捷键、系统通知。
- 需要处理窗口关闭后后台运行。
- 需要考虑 Windows 通知栏和系统语言/主题。

## 第一阶段功能范围

| 模块 | 第一阶段能力 |
|------|--------------|
| Memo | 新建、编辑、删除、搜索、状态、分组、置顶、收藏 |
| 相关人 | 搜索员工、添加相关人、设置 view/edit 权限 |
| 实时同步 | WebSocket 推送 Memo 变更，前端静默刷新 |
| 通知 | 系统弹窗、通知中心、测试通知、通用通知接口 |
| 认证 | Windows 用户识别、设备信任、新设备验证码 |
| 桌面能力 | 托盘、后台运行、快捷键、快速 Memo 窗口 |
| 设置 | 语言、主题、快捷键 |

## 不在第一阶段强做的内容

- IM 聊天
- AI 助手
- 完整企业组织架构管理
- 离线操作队列
- 复杂审批流
- 插件平台

这些能力后续可以接在同一套桌面壳、认证、通知和人员体系之上。

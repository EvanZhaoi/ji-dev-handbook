# 04. UI 系统

## UI 目标

“迹”的 UI 可以沿用当前 UniComm Desktop 的风格：

- 左侧主导航
- 中间 Memo 列表
- 右侧编辑区
- 紧凑、克制、偏 SaaS 工具风
- 支持浅色/深色/跟随系统
- 高 DPI 和 200% 缩放下仍可用

## 色彩和布局

使用 CSS 变量管理主题：

```css
:root {
  --background: 0 0% 100%;
  --foreground: 222 47% 11%;
  --card: 0 0% 100%;
  --border: 214 32% 91%;
  --primary: 226 70% 55%;
}

.dark {
  --background: 222 47% 7%;
  --foreground: 210 40% 96%;
  --card: 222 47% 9%;
  --border: 217 33% 18%;
  --primary: 226 70% 62%;
}
```

建议尺寸：

| 区域 | 尺寸 |
|------|------|
| 左侧导航 | 190px |
| 折叠导航 | 56px |
| Memo 列表 | 280px |
| 标题栏 | 32px |
| 编辑区 padding | 12px |
| 底部操作栏 | 36px |

## 字体

建议内置：

- 中文：阿里巴巴普惠体 3.0
- 日文：Alibaba Sans JP

策略：

- 系统语言是中文时默认中文。
- 系统语言是日文时默认日文。
- 未支持语言默认中文。
- 用户手动修改后以用户设置为准。

## 基础组件优先级

优先使用开源组件减少代码：

| 场景 | 组件 |
|------|------|
| 按钮 | shadcn Button |
| 输入框 | shadcn Input/Textarea |
| 下拉 | Radix Select |
| 弹层 | Radix Popover |
| 右键菜单 | Radix Context Menu |
| 确认弹窗 | Radix Alert Dialog |
| 标签页 | Radix Tabs |
| 图标 | lucide-react |
| 长列表 | TanStack Virtual |

# 10. 开发路线

## Phase 1: 项目骨架

- 创建 `ji-desktop`
- 创建 `ji-server`
- 接入 Tailwind、shadcn 风格组件、Zustand、TanStack Query
- 接入 MySQL、MyBatis-Plus、Sa-Token

## Phase 2: Memo 最小闭环

- Memo 表和接口
- Memo 列表
- Memo 编辑
- 保存状态
- 自动保存与草稿保护

## Phase 3: UI 完善

- 自定义标题栏
- 侧边栏
- 分组/状态/收藏筛选
- 设置页
- 主题切换
- 中日文切换

## Phase 4: 编辑器增强

- Milkdown Crepe
- Markdown 源码
- 图片 base64
- Mermaid 图形
- 长内容滚动

## Phase 5: 桌面能力

- 托盘
- 后台运行
- 全局快捷键
- 快速 Memo 窗口
- 系统通知

## Phase 6: 协作权限

- 成员搜索
- 相关人
- view/edit 权限
- 与我相关
- WebSocket 实时刷新

## Phase 7: 通知平台

- Memo 通知
- 通知中心
- 通用通知表
- 通用通知 API
- 外部系统鉴权

## Phase 8: 生产化

- 真实人员 API
- 邮件验证码
- Redis 多实例广播
- 日志和审计查询
- 离线队列
- 发布安装包
